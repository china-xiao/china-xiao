### 5. DB篇

java后端，不得不打交道的就是DB了，有常见的关系型数据库如MySql；也有一些其他的如文档数据库MongoDB, 时序数据库InfluxDB等；不同的数据库对应不同的应用场景，也因此有不同的使用姿势；总的来说，针对DB相关，需要关注的无法下面一些

- db的相关配置（即环境搭建）
- 增删改查（CURD）
- 数据库事物,分布式的一致性问题
