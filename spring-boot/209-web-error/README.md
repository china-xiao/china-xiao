## 209-web-error
### 项目说明

一个后端应用出现异常怎么办？直接抛出去？层层拦截捕获？

本项目主要介绍如何通过`ControllerAdvice`结合注解`ExceptionHandler`实现一个全局的异常拦截捕获，从而实现全局的异常处理
