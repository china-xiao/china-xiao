## 006-dynamicbean

### 项目说明

Spring-Boot基础之Bean的相关实例工程, 主要包括

- 动态注册Bean实例到Spring容器的几种姿势
- 其他Bean如何使用动态注册的Bean
- 动态注册的Bean如何内部如何注入其他的Bean
- 动态注册和删除bean的实例工程 (参见 `com.git.hui.boot.dynamicbean.example` 包下演示工程)
